# Employee-Attrition-Analysis
Predicting employee turnover using data analysis and interactive dashboards

Using a dataset of approximately 1,470 employees, I explored relationships between job satisfaction, overtime, and attrition outcomes.

---

## 🎯 Objective
To identify key drivers of employee attrition and provide actionable insights for improving retention.

---

## 🛠️ Tools Used
- Tableau
- Power BI
- Excel

---

## 📈 Key Insights
- Employees working overtime are more likely to leave
- Lower job satisfaction is associated with higher attrition
- Certain departments show higher turnover rates
